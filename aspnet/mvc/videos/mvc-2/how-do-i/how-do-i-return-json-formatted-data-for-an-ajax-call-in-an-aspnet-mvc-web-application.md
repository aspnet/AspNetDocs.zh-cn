# <a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>如何实现：在 ASP.NET MVC Web 应用程序中为 AJAX 调用返回 JSON 格式的数据？

作者： [Chris 像素](https://twitter.com/chrispels)

在此视频中，Chris 像素演示了如何使用 JsonResult 和 Json 类将类的实例作为 JSON 格式的数据返回。 首先，创建一个示例 MVC web 应用程序，并为客户信息添加一个控制器和关联的视图。 然后，将创建一个简单的 customer 类作为数据模型的一部分。 然后，将 GetCustomer （）函数添加到客户控制器，以便为 JSON 格式的单个客户检索数据。 最后，将脚本添加到使用 jQuery 对 GetCustomer （）进行 AJAX 调用以获取 JSON 格式的客户数据的客户视图
