# <a name="how-do-i-work-with-model-binders-in-an-mvc-application"></a>如何实现：在 MVC 应用程序中使用模型联编程序？

作者： [Chris 像素](https://twitter.com/chrispels)

在此视频中，Chris 像素演示了如何在 MVC 应用程序中使用模型联编程序将窗体数据绑定到模型中的复杂业务对象。 首先，为 Customer 类创建一个 MVC 应用程序，同时创建一个用于编辑客户数据的控制器和视图。 然后，通过 customer 类演示 MVC 的自动模型绑定。 接下来，将讨论绑定属性，并将其作为一种机制来控制绑定过程的详细信息，以包含和排除复杂对象的特定属性。 最后，讨论如何使用命名空间前缀来分段页面上的表单数据。
