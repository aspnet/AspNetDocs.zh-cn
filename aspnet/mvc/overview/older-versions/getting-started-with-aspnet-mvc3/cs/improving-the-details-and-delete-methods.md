公共语言运行时 (CLR) 需要重载的方法具有一个唯一的签名 （相同的名称，不同的参数列表）。 但是，在这里需要两个 Delete 方法-一个用于 GET-和另一个用于 POST 都需要相同的签名。 （它们都需要接受单个整数作为参数。）
