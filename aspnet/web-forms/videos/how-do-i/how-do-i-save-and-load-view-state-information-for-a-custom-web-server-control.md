# <a name="how-do-i-save-and-load-view-state-information-for-a-custom-web-server-control"></a>[如何实现：]保存和加载自定义 Web 服务器控件的视图状态信息

作者： [Chris 像素](https://twitter.com/chrispels)

在此视频中，Chris 像素将演示如何在自定义用户控件的视图状态中保存和还原信息。 首先，了解如何保存包含 .NET 基元类型（如 String、Int32、Boolean 等）的简单值。然后了解如何保存复杂类型，如包含存储信息的属性的类。 在 ASP.NET 页的生命周期中，还讨论了处理视图状态信息的顺序。
