在本视频中 Chris Pels 将演示如何在 ASP.NET 可以在 web 站点内或跨多个 web 站点中创建自定义用户控件。 启动出学习包括验证的多个页中创建一个用户控件，将使用的基础知识，控制格式设置使用样式元素，并加载/保存数据的用户界面元素。 在整个视频注意事项构建用户控件，可以轻松地维护和使用的多个页面进行讨论。
