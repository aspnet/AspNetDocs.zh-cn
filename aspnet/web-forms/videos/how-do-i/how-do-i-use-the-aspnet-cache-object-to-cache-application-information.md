# <a name="how-do-i-use-the-aspnet-cache-object-to-cache-application-information"></a>[如何实现]：使用 ASP.NET 缓存对象缓存应用程序信息

作者： [Chris 像素](https://twitter.com/chrispels)

在此视频中，Chris 像素演示了如何使用 ASP.NET 缓存对象存储应用程序信息。 示例页是使用简单的用户界面创建的，并添加了一个用 XML 磁盘文件中的数据填充网格的代码。 然后，修改代码，以在最初检索应用程序缓存对象中的数据时将其缓存，并从缓存中读取这些数据以查找后续请求。 最后，添加了一些方法，以利用缓存对象的基于时间的绝对依赖关系。
