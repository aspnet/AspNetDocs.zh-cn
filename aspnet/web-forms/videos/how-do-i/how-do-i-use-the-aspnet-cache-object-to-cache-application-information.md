通过[Chris Pels](https://twitter.com/chrispels)

在此视频的 Chris Pels 中显示了如何使用 ASP.NET 缓存对象来存储应用程序信息。 示例页使用简单的用户界面创建和填充 XML 磁盘文件中的数据网格添加代码。 然后修订代码以在最初检索到和从缓存中读取的后续请求后缓存中的应用程序缓存对象的数据。 最后，将添加方法来使用绝对和可调缓存对象的基于时间依赖关系。
