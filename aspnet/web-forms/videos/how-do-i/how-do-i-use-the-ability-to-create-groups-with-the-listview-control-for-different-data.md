# <a name="how-do-i-use-the-ability-to-create-groups-with-the-listview-control-for-different-data"></a>[如何实现:]使用此功能与不同的数据的 ListView 控件中创建组

通过[Chris Pels](https://twitter.com/chrispels)

在本视频中 Chris Pels 将显示如何在 ListView 控件中显示数据时对项进行分组。 首先，请参阅在 LayoutTemplate 和 ItemTemplate ListView 控件中的项布局的基础知识。 然后，了解 GroupTemplate 与 LayoutTemplate 和 ItemTemplate 内 ListView 来建立一组数据。 请参阅如何使用浮动的 DIV 元素来创建灵活的表格布局，因为任何与 ListView 所用的模板可以包含任何 HTML。 这提供了十分灵活地创建使用 ListView 控件的数据的分组。 最后，请参阅操作中的 ListView 模板以及如何使用 GroupSeparatorTemplate 以及 EmptyItemTemplate 所用的 CSS 样式来增强外观的某些选项。
