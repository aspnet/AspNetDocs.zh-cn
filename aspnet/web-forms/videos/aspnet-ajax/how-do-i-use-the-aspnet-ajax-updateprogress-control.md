# <a name="how-do-i-use-the-aspnet-ajax-updateprogress-control"></a>[如何实现：]使用 ASP.NET AJAX UpdateProgress 控件？

作者： [Joe Stagner](https://github.com/JoeStagner)

在 AJAX 应用程序中，通常会在浏览器发出网络请求与浏览器收到响应的时间之间出现明显的暂停。 为了让用户了解网络会话正在进行，ASP.NET AJAX UpdateProgress 控件通过显示动画图像 "加载" 文本，或在收到网络响应后消失的其他 HTML 来提供即时反馈。
