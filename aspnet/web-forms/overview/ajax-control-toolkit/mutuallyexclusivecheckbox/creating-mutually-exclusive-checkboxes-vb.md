## <a name="overview"></a>概述

可以选择仅一组选项之一，通常用于单选按钮。 还有一个缺点，但所示：选择一个单选按钮组中的后，不能取消选中所有单选按钮。 复选框可以在任何时候是未选中状态，但是不是互相排斥。 本教程提供了最佳的这两种方法： 是互斥的复选框。
