- `<OnClick>` （在控件上的鼠标单击）
- `<OnHoverOut>` （鼠标离开控件）
- `<OnHoverOver>` (鼠标悬停在控件，停止`<OnHoverOut>`动画)

- `<OnMouseOver>` (鼠标悬停在控件，不会停止`<OnMouseOut>`动画)
