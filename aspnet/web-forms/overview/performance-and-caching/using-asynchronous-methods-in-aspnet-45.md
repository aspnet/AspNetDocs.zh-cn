## <a name="processing-asynchronous-requests"></a><span data-ttu-id="606e5-148">处理异步请求</span><span class="sxs-lookup"><span data-stu-id="606e5-148">Processing Asynchronous Requests</span></span>

<span data-ttu-id="606e5-149">在启动时看到大量并发请求的 web 应用程序中，或具有突发负载（其中并发增长突然增加）的 web 应用程序，使 web 服务调用异步会提高应用程序的响应能力。</span><span class="sxs-lookup"><span data-stu-id="606e5-149">In web applications that see a large number of concurrent requests at start-up or has a bursty load (where concurrency increases suddenly), making web service calls asynchronous will increase the responsiveness of your application.</span></span> <span data-ttu-id="606e5-150">异步请求与同步请求所需的处理时间相同。</span><span class="sxs-lookup"><span data-stu-id="606e5-150">An asynchronous request takes the same amount of time to process as a synchronous request.</span></span> <span data-ttu-id="606e5-151">例如，如果某个请求发出的 web 服务调用需要两秒钟才能完成，则该请求将需要两秒钟，无论是同步执行还是异步执行。</span><span class="sxs-lookup"><span data-stu-id="606e5-151">For example, if a request makes a web service call that requires two seconds to complete, the request takes two seconds whether it is performed synchronously or asynchronously.</span></span> <span data-ttu-id="606e5-152">但是，在异步调用期间，线程在等待第一个请求完成时不会被阻止响应其他请求。</span><span class="sxs-lookup"><span data-stu-id="606e5-152">However, during an asynchronous call, a thread is not blocked from responding to other requests while it waits for the first request to complete.</span></span> <span data-ttu-id="606e5-153">因此，当有多个并发请求调用长时间运行的操作时，异步请求会阻止请求队列和线程池的增长。</span><span class="sxs-lookup"><span data-stu-id="606e5-153">Therefore, asynchronous requests prevent request queuing and thread pool growth when there are many concurrent requests that invoke long-running operations.</span></span>

## <a id="ChoosingSyncVasync"></a><span data-ttu-id="606e5-154">选择同步或异步方法</span><span class="sxs-lookup"><span data-stu-id="606e5-154">Choosing Synchronous or Asynchronous Methods</span></span>

<span data-ttu-id="606e5-157">通常，在下列情况下使用同步方法：</span><span class="sxs-lookup"><span data-stu-id="606e5-157">In general, use synchronous methods for the following conditions:</span></span>

<span data-ttu-id="606e5-162">通常，在下列情况下使用异步方法：</span><span class="sxs-lookup"><span data-stu-id="606e5-162">In general, use asynchronous methods for the following conditions:</span></span>

- <span data-ttu-id="606e5-165">并行性比代码的简单性更重要。</span><span class="sxs-lookup"><span data-stu-id="606e5-165">Parallelism is more important than simplicity of code.</span></span>
