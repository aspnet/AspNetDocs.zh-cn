### <a name="url-patterns"></a><span data-ttu-id="cf301-131">URL 模式</span><span class="sxs-lookup"><span data-stu-id="cf301-131">URL Patterns</span></span>

<span data-ttu-id="cf301-132">URL 模式可以包含文字值和变量 （称为 URL 参数） 的占位符。</span><span class="sxs-lookup"><span data-stu-id="cf301-132">A URL pattern can contain literal values and variable placeholders (referred to as URL parameters).</span></span> <span data-ttu-id="cf301-133">文本和占位符中由斜杠分隔的 URL 段的位置 (`/`) 字符。</span><span class="sxs-lookup"><span data-stu-id="cf301-133">The literals and placeholders are located in segments of the URL which are delimited by the slash (`/`) character.</span></span>
