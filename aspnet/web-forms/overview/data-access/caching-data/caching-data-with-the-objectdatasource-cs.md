<span data-ttu-id="af8cd-126">缓存可以极大地提高应用程序 s 整体性能和可伸缩性的生成开销很大的数据并将它的副本存储在可以更有效地访问的位置。</span><span class="sxs-lookup"><span data-stu-id="af8cd-126">Caching can greatly improve an application s overall performance and scalability by taking data that is expensive to generate and storing a copy of it in a location that can be more efficiently accessed.</span></span> <span data-ttu-id="af8cd-127">由于缓存包含只是实际的基础数据的副本，它会过期，或*过时*，如果基础数据发生更改。</span><span class="sxs-lookup"><span data-stu-id="af8cd-127">Since the cache holds just a copy of the actual, underlying data, it can become outdated, or *stale*, if the underlying data changes.</span></span> <span data-ttu-id="af8cd-128">若要应对这种情况，页面开发人员可以指示来缓存项的条件*逐出*从缓存中，使用：</span><span class="sxs-lookup"><span data-stu-id="af8cd-128">To combat this, a page developer can indicate criteria by which the cache item will be *evicted* from the cache, using either:</span></span>
