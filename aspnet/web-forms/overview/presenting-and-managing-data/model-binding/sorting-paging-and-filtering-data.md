## <a name="filter-records-by-user-selection"></a>按用户所选内容筛选记录

模型绑定将添加几个属性使您能够指定如何在模型绑定方法中设置某个参数的值。 这些属性位于**System.Web.ModelBinding**命名空间。 它们包括：

- 控件
- Cookie
- 窗体
- 配置文件
- QueryString
- RouteData
- 会话
- UserProfile
- ViewState
