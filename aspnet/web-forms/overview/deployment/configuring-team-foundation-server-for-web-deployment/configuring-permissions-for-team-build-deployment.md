## <a name="task-overview"></a><span data-ttu-id="90d40-110">任务概述</span><span class="sxs-lookup"><span data-stu-id="90d40-110">Task Overview</span></span>

## <a name="configuring-web-server-permissions"></a><span data-ttu-id="90d40-119">配置 Web 服务器权限</span><span class="sxs-lookup"><span data-stu-id="90d40-119">Configuring Web Server Permissions</span></span>

<span data-ttu-id="90d40-120">如中所述[选择右方法对 Web 部署](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md)，如果你想要将 web 包部署到远程 web 服务器可以使用两种主要方法：</span><span class="sxs-lookup"><span data-stu-id="90d40-120">As described in [Choosing the Right Approach to Web Deployment](../configuring-server-environments-for-web-deployment/choosing-the-right-approach-to-web-deployment.md), there are two main approaches you can use if you want to deploy web packages to a remote web server:</span></span>

- <span data-ttu-id="90d40-121">将从远程位置应用程序部署通过面向*Web 部署代理服务*（也称为远程代理） 目标服务器上。</span><span class="sxs-lookup"><span data-stu-id="90d40-121">Deploy the application from a remote location by targeting the *Web Deployment Agent Service* (also known as the remote agent) on the destination server.</span></span>
- <span data-ttu-id="90d40-122">将从远程位置应用程序部署通过面向*Internet 信息服务*(*IIS) Web 部署处理程序*目标服务器上。</span><span class="sxs-lookup"><span data-stu-id="90d40-122">Deploy the application from a remote location by targeting the *Internet Information Services* (*IIS) Web Deploy Handler* on the destination server.</span></span>

<span data-ttu-id="90d40-123">远程代理在这种情况下具有两个关键限制：</span><span class="sxs-lookup"><span data-stu-id="90d40-123">The remote agent has two key limitations in this case:</span></span>

- <span data-ttu-id="90d40-124">远程代理仅支持 NTLM 身份验证。</span><span class="sxs-lookup"><span data-stu-id="90d40-124">The remote agent supports only NTLM authentication.</span></span> <span data-ttu-id="90d40-125">换而言之，部署必须使用生成服务标识&#x2014;不能模拟另一个帐户。</span><span class="sxs-lookup"><span data-stu-id="90d40-125">In other words, the deployment must use the build service identity&#x2014;you can't impersonate another account.</span></span>
- <span data-ttu-id="90d40-126">若要使用远程代理，执行部署的帐户必须是目标服务器上的管理员。</span><span class="sxs-lookup"><span data-stu-id="90d40-126">To use the remote agent, the account that performs the deployment must be an administrator on the target server.</span></span>

<span data-ttu-id="90d40-127">总之，这些两个限制使远程代理方法不可取的 Team Build 的自动化部署。</span><span class="sxs-lookup"><span data-stu-id="90d40-127">Together, these two limitations make the remote agent approach undesirable for an automated Team Build deployment.</span></span> <span data-ttu-id="90d40-128">若要使用此方法，您需要使生成服务帐户在任何目标 web 服务器上的管理员。</span><span class="sxs-lookup"><span data-stu-id="90d40-128">To use this approach, you'd need to make the build service account an administrator on any target web servers.</span></span>

<span data-ttu-id="90d40-129">与此相反，Web 部署处理程序方法提供了各种优势：</span><span class="sxs-lookup"><span data-stu-id="90d40-129">In contrast, the Web Deploy Handler approach offers various advantages:</span></span>

- <span data-ttu-id="90d40-130">Web 部署处理程序通过 HTTPS，以便您可以将备用帐户的凭据传递给 IIS Web 部署工具 （Web 部署） 支持基本身份验证。</span><span class="sxs-lookup"><span data-stu-id="90d40-130">The Web Deploy Handler supports basic authentication over HTTPS, which allows you to pass the credentials of an alternative account to the IIS Web Deployment Tool (Web Deploy).</span></span>
- <span data-ttu-id="90d40-131">可以配置目标 web 服务器以允许非管理员用户将内容部署到使用 Web 部署处理程序的特定 IIS 网站。</span><span class="sxs-lookup"><span data-stu-id="90d40-131">You can configure target web servers to allow non-administrator users to deploy content to specific IIS websites using the Web Deploy Handler.</span></span>

2. <span data-ttu-id="90d40-135">配置 Web 部署处理程序并为帐户授予所需的权限将内容部署到特定的 IIS 网站，如中所述[用于 Web 部署发布 （Web 部署处理程序） 配置 Web 服务器](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-web-deploy-handler.md)。</span><span class="sxs-lookup"><span data-stu-id="90d40-135">Configure the Web Deploy Handler and grant the account the required permissions to deploy content to a specific IIS website, as described in [Configuring a Web Server for Web Deploy Publishing (Web Deploy Handler)](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-web-deploy-handler.md).</span></span>
3. <span data-ttu-id="90d40-136">调用 Web 部署和目标 Web 部署处理程序，使用基本身份验证并提供域帐户的凭据创建，来执行部署。</span><span class="sxs-lookup"><span data-stu-id="90d40-136">Invoke Web Deploy and target the Web Deploy Handler, using basic authentication and supplying the credentials of the domain account you created, to perform the deployment.</span></span>

<span data-ttu-id="90d40-137">在中[Contact Manager](../web-deployment-in-the-enterprise/the-contact-manager-solution.md)示例解决方案，你指定的身份验证类型 (基本或 NTLM)，Web 部署凭据和特定于环境的项目文件中的终结点地址 （远程代理或 Web 部署处理程序）。</span><span class="sxs-lookup"><span data-stu-id="90d40-137">In the [Contact Manager](../web-deployment-in-the-enterprise/the-contact-manager-solution.md) sample solution, you specify the authentication type (basic or NTLM), the Web Deploy credentials, and the endpoint address (remote agent or Web Deploy Handler) in the environment-specific project file.</span></span> <span data-ttu-id="90d40-138">这些值用于构建和执行项目文件时运行 Web 部署命令。</span><span class="sxs-lookup"><span data-stu-id="90d40-138">These values are used to formulate and run a Web Deploy command when the project file is executed.</span></span> <span data-ttu-id="90d40-139">有关详细信息，请参阅[部署 Web 包](../web-deployment-in-the-enterprise/deploying-web-packages.md)。</span><span class="sxs-lookup"><span data-stu-id="90d40-139">For more information, see [Deploying Web Packages](../web-deployment-in-the-enterprise/deploying-web-packages.md).</span></span>

<span data-ttu-id="90d40-140">有关详细信息配置 Web 部署处理程序，包括如何配置权限，请参阅[用于 Web 部署发布 （Web 部署处理程序） 配置 Web 服务器](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-web-deploy-handler.md)。</span><span class="sxs-lookup"><span data-stu-id="90d40-140">For more information on configuring the Web Deploy Handler, including how to configure permissions, see [Configuring a Web Server for Web Deploy Publishing (Web Deploy Handler)](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-web-deploy-handler.md).</span></span> <span data-ttu-id="90d40-141">有关配置远程代理的详细信息，请参阅[配置 Web 服务器的 Web 部署发布 （远程代理）](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-remote-agent.md)。</span><span class="sxs-lookup"><span data-stu-id="90d40-141">For more information on configuring the remote agent, see [Configuring a Web Server for Web Deploy Publishing (Remote Agent)](../configuring-server-environments-for-web-deployment/configuring-a-web-server-for-web-deploy-publishing-remote-agent.md).</span></span>
