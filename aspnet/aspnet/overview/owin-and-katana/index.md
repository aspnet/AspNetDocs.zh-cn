# <a name="owin-and-katana"></a>OWIN 和 Katana

Katana 是一组灵活的组件，用于生成和托管用于 .NET (OWIN) 的 web 应用的开放 Web 接口。 新开发应使用 [ASP.NET Core](/aspnet/core/introduction-to-aspnet-core)。 这些 `Microsoft.Owin.*` 库不能用于 .Net Core，因为 ASP.NET Core 具有对它们的等效替换项。
