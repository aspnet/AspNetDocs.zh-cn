## <a name="log-for-insight"></a><span data-ttu-id="8048e-175">记录洞察</span><span class="sxs-lookup"><span data-stu-id="8048e-175">Log for insight</span></span>

### <a name="log-in-production"></a><span data-ttu-id="8048e-182">登录生产</span><span class="sxs-lookup"><span data-stu-id="8048e-182">Log in production</span></span>
